Архітектура проєкту та обробка даних

Проєкт організований як пакет structures, що містить модулі для роботи з фундаментальними структурами даних:
structures/: Основний пакет проєкту
hashtable.py: Реалізація хеш-таблиці з подвійним хешуванням, динамічним перехешуванням (Rehashing) та механізмом Lazy Deletion
trees.py: Реалізація Бінарного Дерева Пошуку (BST) для зберігання та сортування транзакцій
__init__.py: Налаштовує зручний експорт класів прямо з пакета
main.py: Точка входу в програму, яка демонструє роботу всіх модулів з використанням кольорового статусу
requirements.txt: Список зовнішніх залежностей проєкту
history.log: Файл текстового логування операцій з механізмом автоматичної ротації
